3-Month Inflation Forecast

Code 
# Chart 3: Recent YoY inflation + ARIMA model forecast for the next 3 months.
# Shaded bands represent 80% and 95% confidence intervals.
# history_n = 60 shows the last 5 years for readability.
# To use this chart alone in a presentation: copy just this chunk.
p_cpi_forecast <- plot_cpi_forecast(cpi, history_n = 60)
maybe_interactive(p_cpi_forecast, INTERACTIVE)
Note

Data source: U.S. Bureau of Labor Statistics (BLS), CPI-U All Items Seasonally Adjusted (series CUSR0000SA0). Released monthly, typically in the second week of the following month. Forecast produced by an ARIMA model fit to the YoY % change series; confidence intervals reflect model uncertainty only and do not incorporate external shocks.
