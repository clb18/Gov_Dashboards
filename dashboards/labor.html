3-Month Unemployment Forecast

Code 
# Chart 2: Recent unemployment + ARIMA model forecast for the next 3 months.
# history_n = 60 shows the last 5 years for readability.
# Shaded bands represent 80% and 95% confidence intervals.
# To use this chart alone in a presentation: copy just this chunk.
p_unrate_forecast <- plot_unrate_forecast(labor, history_n = 60)
maybe_interactive(p_unrate_forecast, INTERACTIVE)
Note

Data source: U.S. Bureau of Labor Statistics (BLS), Current Population Survey (CPS). U-3 Unemployment Rate, Seasonally Adjusted (series LNS14000000). Released monthly on the first Friday of the following month. Recession shading based on NBER Business Cycle Dating Committee designations. Forecast produced by an ARIMA model; confidence intervals reflect model uncertainty only.
